\#BW Widgets Manager provides a simple interface to deactive unused widgets, cleaning up the widgets admin page

== Description ==

The admin widgets page can get very cluttered with too many widgets, many of which you will never use. **\#BW Widgets Manager** provides an interface to selectively deactivate unused widgets, cleaning up the widgets admin page.

== Installation ==

1. Upload bw-widgets folder to your /wp-content/plugins/ directory
2. Activate the plugin in Wordpress admin
3. Deactive widgets from the plugin admin page, under #BW Options -> Widgets Manager
